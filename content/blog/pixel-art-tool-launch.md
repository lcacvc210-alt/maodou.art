## 🎉 又一个新工具上线了！

今天下午，**像素图转换器**正式在 maodou.art 上线！

这是继配色灵感库之后，我们上线的第二个创意工具。

---

## 🕹️ 什么是像素图转换器？

简单来说，它能把你的照片变成**复古像素风**效果。

就像这样：

```
原图 → 像素化处理 → 像素风头像
```

**核心功能：**
- ✅ 上传图片，一键生成像素效果
- ✅ 支持 4×4 到 64×64 密度调节
- ✅ 自动生成 35×35 推荐头像尺寸
- ✅ 实时预览，即时下载 PNG

## 🎨 为什么做这个工具？

### 1️⃣ 像素风的魅力

像素艺术（Pixel Art）是一种独特的美学风格：

- **复古感**：让人想起 8-bit 游戏时代
- **简约美**：用有限的像素表达丰富的内容
- **个性化**：每张图片都有独特的像素质感

### 2️⃣ 实用场景

像素风头像特别适合：
- 社交媒体头像（Twitter、Discord、GitHub）
- 游戏角色图标
- 极简风格设计
- 创意项目展示

### 3️⃣ 技术探索

这个工具背后是有趣的图像处理算法：
- 网格切分
- 颜色取样
- 平均色计算
- 色块填充

## 🛠️ 技术实现（给感兴趣的你）

### 核心算法

```javascript
// 简化版伪代码
function pixelate(image, gridSize) {
  // 1. 计算像素块大小
  const blockSize = imageWidth / gridSize
  
  // 2. 遍历每个像素块
  for (let y = 0; y < imageHeight; y += blockSize) {
    for (let x = 0; x < imageWidth; x += blockSize) {
      // 3. 获取块内所有像素
      const pixels = getPixels(x, y, blockSize, blockSize)
      
      // 4. 计算平均色
      const avgColor = average(pixels)
      
      // 5. 填充整个块为平均色
      fillRect(x, y, blockSize, blockSize, avgColor)
    }
  }
}
```

### 关键技术点

| 技术 | 说明 |
|------|------|
| Canvas API | 图像处理和绘制 |
| getImageData | 获取像素数据 |
| 平均色算法 | RGB 各通道取平均值 |
| 跨域处理 | CORS 配置 |

### 性能优化

- 使用离屏 Canvas 避免重复绘制
- 异步处理防止 UI 阻塞
- 预生成演示图提升首屏体验

## 📖 使用教程

### 第一步：打开工具

访问：**https://maodou.art/tools/pixel-art**

### 第二步：上传图片

点击「选择图片」或拖拽上传：
- 支持 JPG、PNG、GIF 格式
- 建议图片尺寸：500×500 以上
- 文件大小：最大 10MB

### 第三步：调节密度

上传后自动生成 **35×35** 默认效果（推荐头像尺寸）

你也可以手动调节：
- **4×4 - 8×8**：强像素感，适合图标
- **16×16 - 32×32**：平衡细节，通用场景
- **48×48 - 64×64**：细节丰富，轻微像素化

### 第四步：下载保存

点击「下载」按钮，保存 PNG 格式：
- 保持原图尺寸
- 透明背景支持
- 无损压缩

## 💡 使用建议

### 适合的图片类型

✅ **人像照片**：生成像素风头像效果最佳
✅ **风景照片**：选择高密度（32×32 以上）
✅ **卡通/插画**：低密度就有很好效果
✅ **Logo/图标**：4×4 - 8×8 极简风格

### 不适合的图片类型

❌ **文字截图**：像素化后文字难以辨认
❌ **细节复杂**：如密集的建筑群
❌ **低分辨率**：本身就很模糊的图片

### 参数推荐

| 用途 | 推荐密度 | 说明 |
|------|---------|------|
| 社交媒体头像 | 35×35 | 平衡识别度和像素感 |
| 游戏图标 | 16×16 - 24×24 | 强像素感 |
| 艺术创作 | 8×8 - 12×12 | 极简风格 |
| 照片处理 | 48×48 - 64×64 | 保留更多细节 |
