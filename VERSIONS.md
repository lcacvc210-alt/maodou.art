## 版本规范

### 版本号格式
```
主版本。次版本。修订版本
  ↓      ↓      ↓
  X  .   Y   .  Z
```

### 更新规则
- **日更版本**：每天结束时的版本迭代 (v1.0 → v1.1 → v1.2)
- **大版本**：重大功能更新时升级主版本 (v1.x → v2.0)
- **基准版本**：v1.0 (2026-03-12) - 网站正式上线

### 推送流程
```
本地修改 → 测试 → 提交 Git → 晚上总结 → 版本号 +1 → 询问超哥 → 推送公网
```
